# Bibliotecas Utilizadas

## Backend
- **Express**: Framework web para Node.js (utilizado em server.js)
- **Mongoose**: ODM para MongoDB
- **bcryptjs**: Hash de senhas
- **cors**: Middleware para CORS
- **nodemailer**: Envio de e-mails
- **dotenv**: Gerenciamento de variáveis de ambiente
- **jsonwebtoken**: Autenticação JWT
- **TypeORM**: ORM para bancos relacionais (usado no módulo NestJS)
- **@nestjs/core, @nestjs/common, @nestjs/typeorm**: Framework NestJS e integrações

## Frontend (Client/Admin)
- **Chart.js**: Geração de gráficos
- **Font Awesome**: Ícones
- **JavaScript Puro**: Não há uso de frameworks modernos (React, Vue, Angular) no frontend analisado
